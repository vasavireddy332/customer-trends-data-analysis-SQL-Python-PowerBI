# Customer-trends-data-analysis-SQL-Python-PowerBI
This project is a complete, industry-standard, end-to-end data analytics workflow designed to mirror the real responsibilities of professional data analysts in modern business environments. It covers all critical stages of analysis — from raw data exploration and SQL querying to Python-based EDA and interactive Power BI dashboards.The goal is to uncover actionable insights into customer shopping behavior: who buys what, how much they spend, what influences their decisions, and how businesses can target them more effectively.This project is a complete, industry-standard, end-to-end data analytics workflow designed to mirror the real responsibilities of professional data analysts in modern business environments.  # SQL — Business Questions Answered
1.Total revenue by gender
2.High-value customers who used discounts
3.Top 5 products by average review rating
4.Standard vs. Express shipping spend comparison
5.Subscriber vs. non-subscriber revenue analysis
6.Products with highest discount usage rates
7.Customer segmentation — New / Returning / Loyal (CTE)
8.Top 3 products per category (Window Function)
9.Repeat buyers and subscription likelihood
10.Revenue contribution by age group
# Power BI Dashboard Covers
1.Revenue by gender, age group & category
2.Subscriber vs. non-subscriber spend
3.Discount usage impact
4.Product ratings & performance
5.Customer segmentation visuals
Download the  [customer_behavior_dashboard.pbix] and open with Power BI Desktop
#  🚀 How to Run
SQL: Import the CSV into PostgreSQL (table: customer) → run customer_behavior_sql_queries.sql
Power BI: Open .pbix in Power BI Desktop, re-link CSV if prompted
